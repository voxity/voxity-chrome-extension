Dépendances

Le plugin dépend de plusieurs outils libre, AngularJs, et c'est extensions Angular Route, Angular Sanitize, Angular Animate, Angular i18n, Angular Ui Bootstrap. Ainsi que de Bootstrap3 et Font Awesome.
